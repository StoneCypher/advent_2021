getting started

One time instructions:

  1. Fork the repo
  2. Turn github pages on on your fork, for main branch, /docs directory
  3. Download your fork
  4. Go into %/src/ts/config.ts and set your things
  5. Run a build npm run build
  6. Commit and push the results

Should be ready.



to run

npm run build



to run just one day

npm run day 6



to add a day

  1. copy five files from day 0 to the new day number
    • %/src/ts/day0.ts
    • %/src/ts/day0.spec.ts
    • %/src/ts/data/0_example.txt
    • %/src/ts/data/0_example_solution.txt
    • %/src/ts/data/0_problem.txt
  2. modify %/src/ts/days.ts to add the new day to the list
  3. replace the contents of the data files as appropriate
  4. do your work in dayN.ts
  5. do any day-specific tests in dayN.spec.ts
  6. run a build
